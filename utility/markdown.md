# Markdown常用格式

## 标题  
【语法】  
\# This is an H1  
\## This is an H2  
\### This is an H3  
\#### This is an H4  
\##### This is an H5  
\###### This is an H6  

【效果】  
# This is an H1
## This is an H2
### This is an H3
#### This is an H4
##### This is an H5
###### This is an H6

## 强调  
【语法】  
\*强调*  
\*\*加重强调**  
\*\*\*特别强调***  
 
【效果】  
*强调*  
**加重强调**  
***特别强调***  

## 代码块
【语法】  
前面加4个空格

【效果】  

    sample code

## 引用
【语法】  
前面加“\>”

【效果】  
>sample code

## 链接
【语法】  
\[链接文字](链接地址)

【效果】  
[图标](https://avatars0.githubusercontent.com/u/24515063?v=3&s=96)
 
## 分割线
【语法】  
在单独一行里输入3个或以上的短横线、星号或者下划线实现

【效果】  
* * *  
- - -  
_ _ _  
 
## 无序列表
【语法】  
前面加“\* ”、“\+ ”、“\- ”

【效果】  
* Item1
* Item2
* Item3

## 有序列表
【语法】  
前面加“\<number>. ”，同时后方文字要加任意一种强调

【效果】  

1. *Itme1*
2. **Item2**
3. ***Item3***
